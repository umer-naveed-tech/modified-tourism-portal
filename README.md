# 🏨 Ahmed Travels - Tourism Portal

A professional tourism portal designed for Saudi Arabia, allowing users to browse hotels, view rooms, and manage bookings.

## ✨ Features

- 🔐 User Login & Signup (Visitor & Agent roles)
- 🏨 Hotel Listings (Mecca & Madinah)
- 🛏️ Room Management (Separate, Double, Triple, Quad)
- 📅 Booking System
- 🎨 Responsive & Professional Design
- 🗄️ MySQL Database Integration

## 🛠️ Technologies Used

- PHP (Backend)
- MySQL (Database)
- HTML, CSS (Frontend)
- JavaScript (Interactivity)

## 🚀 How to Run

1. Clone the repository
2. Import database from `database.sql`
3. Update `includes/config.php` with your MySQL credentials
4. Run `php -S localhost:8000`
5. Open `http://localhost:8000`
